# Save

Save the internal state of a list of firemans and encode it in JSON.

**URL** : `/save/:ids`

**URL Parameters** : `ids=[list_integer]` where `ids` is  a list of fireman ids regisered on the expert system

**Method** : `GET`

**Auth required** : NO

## Success Response

**Code** : `200 OK`

**Content example** :

```json
{
    "": ""
}
```
